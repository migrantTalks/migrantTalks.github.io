Redis 安装

四月 19, 2019
  • redis

Redis是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API

前期准备

  • VMware Workstation 是一款功能强大的桌面虚拟计算机软件，提供用户可在单一的桌面上同时运行不同的操作系统，和进行开发、测试 、部署新的应用程序的最佳解决方案

  • Centos7 是一个基于 Red Hat Linux 提供的可自由使用源代码的企业级Linux发行版本

  • Xshell 是一个强大的安全终端模拟软件，它支持 SSH1, SSH2, 以及 Microsoft Windows 平台的TELNET 协议

  • FileZilla 是一个免费开源的 FTP 软件，分为客户端版本和服务器版本，具备所有的FTP软件功能

下载Redis

通过 https://redis.io/download 下载对应版本

安装步骤

  • 安装依赖编译环境 gcc

  • 通过 FileZilla Client 把下载的 redis 压缩包( redis-5.0.4.tar.gz )传输到 Centos7 上

  • 解压文件 redis-5.0.4.tar.gz

  • 进入 redis 解压后的文件目录 ，执行 make
  • 把 redis 安装 到 /usr/local/redis 目录下
  • 把 redis-5.0.4 的 redis.conf 文件，复制到 /usr/local/redis
  • 在 /usr/local/redis 目录下，修改运行模式，把 daemonize no 改为 daemonize yes
  • 在 /usr/local/redis 目录下，启动 redis
  • 查看 redis 进程
  • 在 /usr/local/redis 目录下，停止 redis 服务
  • 在 /usr/local/redis 目录下，进入 redis 服务

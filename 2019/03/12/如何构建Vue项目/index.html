如何构建Vue项目

三月 12, 2019
  • node
  • vue

Vue 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

安装node.js开发环境

  • 去网站 https://nodejs.org/en/download/, 下载对应的安装程序，并安装
  • 配置”NODE_HOME”为：”D:\nodejs”
  • 配置”NODE_PATH”为：%NODE_HOME%\node_modules
  • 配置Path，追加 ;%NODE_HOME%\;%NODE_PATH%\;

安装Vue环境

  • 安装cnpm淘宝镜像
  • 配置包安装目录
  • 安装Vue脚手架
  • 构建项目
  • 下载项目组件
  • 运行项目
  • 访问项目

http://localhost:18099

  • 修改主机、端口号(config/index.js)

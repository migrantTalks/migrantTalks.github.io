如何构建Vue项目

三月 12, 2019
  • vue

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

学习Vue的前提需要安装Vue的相关开发环境，通过查阅他人资料并自己安装后，整理出一篇安装文档，从而加深学习。

安装node.js开发环境

  • 配置”NODE_HOME”为：”D:\nodejs”
  • 配置”NODE_PATH”为：%NODE_HOME%\node_modules
  • 配置Path，追加 ;%NODE_HOME%\;%NODE_PATH%\;

node -v 或 npm-v

安装Vue环境

  • 配置包安装目录

npm config set prefix “node路径/node_global” 设置全局包安装的地方
npm config set cache “node路径/node_cache” 设置node缓存保存的地方
配置Path，追加 ;%NODE_HOME%\node_global;

  • 安装Vue脚手架

npm install -g webpack
npm install -g vue-cli

  • 构建项目

vue init webpack admin

  • 项目组件

npm install

  • 运行项目

npm run dev

  • 访问项目

http://localhost:8080

  • 修改主机、端口号(config/index.js)

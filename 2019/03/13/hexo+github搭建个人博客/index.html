Hexo+GitHub 搭建个人博客

三月 13, 2019
  • Hexo

由于想通过写博客，记录自己平时的一些技术笔记，可提升自己的写作、总结能力。然后查询了一些资料后，最终考虑通过 Hexo + GitHub 搭建个人博客。

安装node.js

  • 去网站 https://nodejs.org/en/download/, 下载对应的安装程序，并安装
  • 配置”NODE_HOME”为：”D:\nodejs”
  • 配置”NODE_PATH”为：%NODE_HOME%\node_modules
  • 配置Path，追加 ;%NODE_HOME%\;%NODE_PATH%\;

安装Git

  • 去网站 https://git-scm.com/download/win, 下载对应的安装程序，并安装
  • 配置用户信息

配置GitHub

建立与你用户名对应的仓库，仓库名必须为【your_user_name.github.io】

安装Hexo

  • 通过npm下载hexo
  • 初始化一个blog
  • 运行hexo
  • 本地访问hexo

    http://localhost:4000/

Hexo配置

  • 博客信息配置，可参考 https://hexo.io/zh-cn/api/
  • 主题配置

1、进入themes文件夹，通过网站 https://hexo.io/themes/index.html 下载自己喜欢的theme

2、修改_config.yml配置主题

3、再次启动hexo，主题风格已换成自己喜欢的了

编辑文章

在 blog\source_posts下会生成一个helloWord.md文件

部署到GitHub

  • 修改_config.yml配置，部署到GitHub

  • 安装 hexo-deployer-git

  • 部署

  • 访问个人博客
    https://migrantalks.github.io/
